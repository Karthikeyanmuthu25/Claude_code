## What worked well

- Splitting the setup guide (`INSTALL.md`) from the project context file (`CLAUDE.md`) — clear separation of concerns
- The `claude doctor` command is extremely useful for diagnosing setup issues
- Native installer (`curl -fsSL https://claude.ai/install.sh | bash`) is significantly simpler than npm method

## What didn't work / Gotchas

- Claude Code does NOT auto-detect Cursor IDE — manual VSIX install is required
- `sudo npm install -g` causes permission issues — avoid this
- Node.js is NOT required for the native installer (contrary to many guides online)
- Windows CMD/PowerShell DO work with Claude Code (with Git for Windows installed)
