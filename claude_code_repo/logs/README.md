# Activity Logs

This folder tracks day-by-day Claude Code activities, experiments, and discoveries.

---

## How to add a log entry

1. Create a new file: `YYYY-MM-DD.md` (e.g. `2026-03-06.md`)
2. Use the template below
3. Commit with message: `logs: add YYYY-MM-DD activity`

---

## Log Template

Copy this into your new file:

```markdown
# Log — YYYY-MM-DD

## What I worked on
<!-- Brief description of the task or experiment -->

## Claude Code usage
<!-- How did you use Claude Code? What commands, modes, or slash commands? -->

## What worked well
<!-- Wins, effective prompts, useful patterns discovered -->

## What didn't work / Gotchas
<!-- Failures, unexpected behaviour, limitations hit -->

## Configuration changes
<!-- Any updates to CLAUDE.md, .claudeignore, settings.json -->

## Next steps
<!-- What to try or follow up on next session -->
```
