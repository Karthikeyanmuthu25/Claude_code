# Slash Commands Reference

All Claude Code slash commands, organised by category.

---

## Session

| Command | Description |
|---------|-------------|
| `/clear` | Clear conversation, start fresh |
| `/compact` | Compress context to free up the context window |
| `/context` | View current context window usage |
| `/cost` | Show token usage and estimated cost |
| `/quit` / `/exit` | Exit Claude Code |

## Configuration & Memory

| Command | Description |
|---------|-------------|
| `/init` | Auto-generate CLAUDE.md for current project |
| `/memory` | Open and edit CLAUDE.md memory files |
| `/config` | Open full settings interface |
| `/model` | Switch between Claude models |
| `/permissions` | View and edit tool permissions |

## Tools & Integration

| Command | Description |
|---------|-------------|
| `/ide` | Connect to current IDE (VS Code, Cursor) |
| `/terminal-setup` | Configure Shift+Enter for multi-line input |
| `/mcp` | Manage MCP (Model Context Protocol) server connections |
| `/install-github-app` | Enable automatic PR reviews |
| `/plan` | Enter Plan Mode |

## Other

| Command | Description |
|---------|-------------|
| `/help` | Show all available commands |
| `/vim` | Enable Vim editing mode |
| `/bug` | Report a bug from within Claude Code |
| `/review` | Trigger a code review |
| `/rewind` | Undo the last action |
| `/usage` | Show plan usage and limits |

---

> Run `/help` inside any session for the most up-to-date command list.
