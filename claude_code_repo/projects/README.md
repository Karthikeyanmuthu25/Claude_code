# Projects

This folder contains structured experiments and real project integrations using Claude Code.

---

## How to add a project

1. Create a subfolder: `projects/your-project-name/`
2. Add a `README.md` inside it using the template below
3. Document what Claude Code did, what worked, and what configuration you used
4. Update the index table below

---

## Project Template

```markdown
# Project: [Name]

## Overview
<!-- What is this project? What problem does it solve? -->

## Stack
<!-- Languages, frameworks, tools -->

## How Claude Code was used
<!-- Tasks delegated to Claude Code, modes used, slash commands, CLAUDE.md setup -->

## Key findings
<!-- What worked well, what surprised you, performance notes -->

## Configuration
<!-- Link to or inline the CLAUDE.md and .claudeignore used -->

## Outcome
<!-- Did it work? What would you do differently? -->
```
